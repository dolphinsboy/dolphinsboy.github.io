在Intellij IDEA中配置golang

发表于 2015-07-22 | 分类于 Go

1、环境搭建

参照下面的博客完成在Intellij IDEA中配置golang
IntelliJ IDEA 14安装Golang插件go-lang-idea-plugin。

由于系统上之前已经安装好go和IDEA，现在将两者结合起来，本博客记录整个配置的过程。

1.1 下载插件

在https://github.com/go-lang-plugin-org/go-lang-idea-plugin上下载插件。

1.2 配置生成intellij-go.jar包

1、将下载的go-lang-idea-plugin压缩包解压，IDEA点击File -> open 打开该工程。

2、点击File -> Project Structure 打开项目结构配置窗口：

3、设置Project SDK
点击File -> Project Structure ->Project，设置Project SDK为刚刚设置好的IDEA sdk

4、编译生成jar包
点击Build->Prepare All Plugins For Deployment，选择intellij-go.jar包

1.3 安装插件

在Perference中选择Plugins

手动安装intellij-go.jar包

1.4 设置Go SDK


选择go的HOME
