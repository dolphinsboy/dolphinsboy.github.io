L190 Reverse Bits

发表于 2017-12-08

题目描述

解题思路

解题思路1——使用list记录然后reverse重新计算10进制数值

  • 先计算出n对应的二进制数保存到list中，对于不满足32位的数设置0
  • 将list翻转后，再重新计算10进制数值

不使用List

  • 先数值计算
  • 然后判断位数是否小于32，进行左移位

使用bin和zfill函数

  • 使用bin获取int对应的二进制
  • 使用zfill补齐零
  • 使用int计算二进制对应的十进制
